### Project Overview

This project is a chat-like application where I utilize Firebase as the backend for data storage. It also employs P2P connections to enable video calls.

![Imgur](../images/Connect2.png)

#### Technologies Used

- React
- Vite
- Firebase
- TypeScript
- Tailwind CSS
- Zustand
